  # IOC Cybersecurity Dashboard

## Descripción

Dashboard desarrollado en Power BI para analizar Indicadores de Compromiso (IoC) de ciberseguridad.

## Dataset

- Indicadores IoC
- IP Address
- FQDN
- URL
- MD5
- SHA-1
- SHA-256

## Objetivos

- Analizar tipos de IoC.
- Identificar actores de amenaza.
- Analizar fuentes de inteligencia.
- Visualizar tendencias temporales.

## Herramientas

- Power BI
- Power Query
- CSV
- GitHub
# IOC Cybersecurity Dashboard



## Visualizaciones

- Distribución por Tipo
- Top Adversarios
- Fuentes de Inteligencia
- Tendencia Temporal

## Hallazgos

- FQDN fue el tipo más frecuente.
- Google Threat Intelligence fue una de las principales fuentes.
- Se identificaron actores como FIN7 y APT44.
