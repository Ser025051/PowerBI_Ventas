  # IOC Cybersecurity Dashboard

## Descripción

Dashboard desarrollado en Power BI para analizar Indicadores de Compromiso (IoC) de ciberseguridad.

## Dataset

- Indicadores IoC
- IP Address
- FQDN
- URL
- MD5
- SHA-1
- SHA-256

## Objetivos

- Analizar tipos de IoC.
- Identificar actores de amenaza.
- Analizar fuentes de inteligencia.
- Visualizar tendencias temporales.

## Herramientas

- Power BI
- Power Query
- CSV
- GitHub
