# Dashboard de Ventas - Power BI

## Descripción

Proyecto de análisis de ventas desarrollado con Power BI.

## Objetivos

- Analizar las ventas por periodo.
- Identificar los productos más vendidos.
- Detectar tendencias de crecimiento.
- Visualizar indicadores clave de negocio.

## Herramientas

- Power BI
- SQL
- Excel

## Indicadores

- Ventas totales
- Cantidad de pedidos
- Ticket promedio
- Top productos
- Ventas por mes
